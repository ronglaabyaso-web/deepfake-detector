# วิธีทำให้ "ตัวช่วยจับคลิปปลอม" เป็นแอปติดตั้งได้

แอปนี้เป็น **PWA (Progressive Web App)** — เปิดผ่านเบราว์เซอร์แล้วกด "ติดตั้ง" ได้เลย ไม่ต้องผ่าน App Store

## 1. ติดตั้งบนมือถือ (ใช้ในวง LAN เดียวกัน)

1. รันเซิร์ฟเวอร์บนคอมพิวเตอร์: `cd backend && python run.py`
2. ดู IP ของคอม (Windows: `ipconfig` ดู IPv4 เช่น `192.168.1.50`)
3. บนมือถือ (Wi-Fi เดียวกัน) เปิดเบราว์เซอร์ไปที่ `http://192.168.1.50:8000`
4. **Android (Chrome):** เมนู ⋮ → "เพิ่มลงในหน้าจอหลัก" / "ติดตั้งแอป"
5. **iPhone (Safari):** ปุ่มแชร์ □↑ → "เพิ่มลงในหน้าจอโฮม"

## 2. เปิดให้เข้าจากอินเทอร์เน็ตฟรี (cloudflared tunnel)

เหมาะสำหรับเดโม/ให้กรรมการสแกน QR — ฟรี ไม่ต้องมีโดเมน ไม่ต้องเปิด port

```bash
# ติดตั้ง cloudflared (ครั้งเดียว)
# Windows: winget install Cloudflare.cloudflared
# macOS:   brew install cloudflared

# รันเซิร์ฟเวอร์ก่อน แล้วเปิด tunnel
cloudflared tunnel --url http://localhost:8000
```

จะได้ลิงก์แบบ `https://xxxx.trycloudflare.com` — ใช้ได้ทั้ง HTTPS (จำเป็นสำหรับติดตั้ง PWA บน Android)
เอาลิงก์ไปทำ QR code ฟรีได้ที่หลายเว็บ (เช่นค้นหา "QR code generator")

> หมายเหตุ: ลิงก์ trycloudflare เปลี่ยนทุกครั้งที่รันใหม่ — สร้าง QR ใหม่ก่อนเดโมทุกครั้ง

## 3. Deploy frontend ฟรี (ทางเลือก)

ถ้าอยากให้หน้าเว็บอยู่บนลิงก์ถาวร:

1. อัปโหลดโฟลเดอร์ `frontend/` ขึ้น GitHub Pages / Cloudflare Pages / Netlify (ฟรีทั้งหมด)
2. เปิด backend ด้วย cloudflared tunnel (ข้อ 2)
3. ในแอป หน้า **"ของฉัน" → ตั้งค่าที่อยู่เซิร์ฟเวอร์** ใส่ลิงก์ tunnel ของ backend

frontend ตรวจ origin เองอัตโนมัติ: ถ้าเปิดจากเซิร์ฟเวอร์เดียวกับ backend จะเรียก API ที่เดียวกันทันที ไม่ต้องตั้งค่า
